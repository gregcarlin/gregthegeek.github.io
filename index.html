Student

  • I attended a competitive enrollment program at Columbia University in the summer of 2013, where I took the course “Computer Graphics through Game Programming”.

  • I am a member of the Mu Alpha Theta math honor society.
  • I am a member of the World Language Honor Society.

Developer

  • I designed, built, and currently maintain and enhance My Trust Co, a bespoke website dedicated to enabling individuals to organize trusts. This is an ongoing project that is organized by a local couple.
  • I created this website with an original design.
  • I participated in the HackBCA competition with a peer. We created Codium, a website designed to allow educators to teach classes of students how to program over long distances via video chat and live collaboration. We placed in the top 20 and won the best use of the TokBox API.
  • My team and I won second place at PennApps X, another hackathon. We created MagicBoard, which connected a BoostedBoard with a Myo gesture controlled armband. This allowed the user to control the speed of the board by rotating his or her arm.
  • My team and I created LetsGit, a cloud-based git client, at HackGenY. It allows programmers to work on projects without downloading git software. This is useful for situations in which one does not have admin access to a computer. It also helps designers collaborate with programmers, as they can directly modify a project's files without having to learn git.
  • My team and I created Airduino at MHacks V. It allows students to change variables in Arduino software, allowing them to get interested in hardware despite not knowing how to code.
  • I am currently working with a peer on jGrader, a tool to help AP Computer Science teachers easily collect and grade homework.
  • I created a Neural Network with two peers that was able to recognize handwritten digits with 98% accuracy.
  • In around the eighth to tenth grades I was on the developer team for the CanaryMod project, a server mod for Minecraft.
  • I work on small projects all the time, some of which can be found on my Github.
